About

This site is a minimal, static transparency portal for the Central Student Government — Imus Campus. It is intentionally simple so content can be updated easily by editing the data arrays in Static/main.js or by replacing files.

For hosting, place this folder on any static web server (Apache, nginx, GitHub Pages, etc.).
